## 阶段产出检查清单

### DISCUSSION 阶段
- [ ] 所有Agent理解了相同的需求
- [ ] 核心功能列表已确认
- [ ] 边界已划定
- [ ] 疑问点已解决

### DESIGN 阶段
- [ ] 技术栈已选定
- [ ] 架构图已绘制
- [ ] API设计已明确
- [ ] 分工方案已通过

### DEVELOP 阶段
- [ ] 各Agent按分工执行
- [ ] 产出符合设计要求
- [ ] 遇到阻塞及时上报

### INTEGRATE 阶段
- [ ] 前端后端接口对接
- [ ] 功能联调通过
- [ ] 无严重阻塞问题

### DELIVER 阶段
- [ ] 功能完整性验收
- [ ] 文档齐全
- [ ] 部署就绪
